Animations

Handling simple animations with CSS is easy. Utilizing the transition property, we can select different CSS properties that we want to animate. In this case, we are looking at the boxes below. We will focus on transitioning the left and transform properties which at the sitting state are 0% and translateX(0%) respectively. Using JavaScript, we are going to toggle on and off a class called right which has alternative CSS that changes our properties to left: 100%; and transform: translateX(100%);

Finally, setting the transition property, we can tell the boxes to take a certain amount of time to complete their change in properties. You can even specify different transition timings for each property by specifying the name of the property, time, and easing.

Toggling Darkmode

The button below is a standard implementation of toggling darkmode. Simply put, we are adding a class darkmode to the body element when dark mode is enabled, and removing it when disabled. This allows us to create CSS selectors that have a specificity that looks for elements that are children of a body tag with the class darkmode.

For example, we might have the selector body.darkmode button which translates to "In a body tag with the class darkmode, find all buttons and style them the following way...".
